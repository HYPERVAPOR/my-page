# 项目说明

本项目 fork 自 [soumyajit4419/Portfolio](https://github.com/soumyajit4419/Portfolio)，在原有基础上进行了功能扩展和优化。

## 新增特性

- 支持中英文切换，方便不同语言用户浏览
- 支持中英文简历切换，展示不同语言版本的简历内容

## 快速开始

```bash
# 克隆项目代码
git clone https://github.com/HYPERVAPOR/my-page
cd my-page

# 安装依赖
npm install

# 启动开发服务器
npm run start

# 构建生产版本
npm run build
```
